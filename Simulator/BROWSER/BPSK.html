Overview

The balanced modulator with the carrier sine wave as one input and the binary sequence as the second input makes up the Binary Phase Shift Keying modulator. The modulation is performed with a balanced modulator, which multiplies the two input signals. The phase reversal for a zero binary input is 0°, while for high binary input, it is 180°. The direct input carrier or the inverted 180°phase shifted input carrier, a function of the data signal, will be the modulator's output sine wave.

The demodulator is composed of a mixer with a local oscillator circuit, a bandpass filter, and a two-input detector circuit. With the help of the mixer circuit and the bandpass filter, the first stage of demodulation is performed by recovering the band-limited message signal. It is possible to obtain a band-limited baseband signal. The binary message bitstream is regenerated using this signal. The bit clock rate is required at the detector circuit in the following stage of demodulation to produce the original binary message signal. Bit clock regeneration is simplified if the bit rate is a sub-multiple of the carrier frequency.
