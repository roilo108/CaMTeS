Overview

In digital companding, it is now paired with digital compression instead of having the ADC at the transmitter alone. The transmitter consists of a 12-bit ADC and a digital compressor. The input is an analog signal in the transmitter, and the output is an 8-bit compressed code. It uses a 12-bit ADC instead of 8-bit in the basic PCM. The analog signal is sampled, quantized, and encoded in this stage. The encoding in the ADC is a 12-bit folded binary code and is later applied to the digital compression process. The digital compression process will convert the 12-bit folded binary code into an 8-bit compressed code called the transmitted PCM code. At the receiver, the reverse process is done. The 8-bit compressed code is applied at the digital expander to expand the 8-bit compressed code into a 12-bit expanded code. It will then undergo the process of 12-bit digital-to-analog conversion. The 12-bit DAC will reconstruct an analog signal based on the latter code.
