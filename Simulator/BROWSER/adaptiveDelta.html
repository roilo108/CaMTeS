Overview

Adaptive Delta Modulation is similar to Delta Modulation, except that the step size is variable according to the input signal. The analog signal is sampled at a rate higher than the Nyquist rate. The current sample is compared with the encoded value of the previous sample. The result of the comparison is encoded with one bit. Suppose the current sample is greater than or equal to the encoded value of the previous sample plus the stepsize. In that case, a 1 is transmitted; otherwise, 0. If a 0 is transmitted, the encoded value of the current sample is equal to the encoded value of the previous sample minus the stepsize. The transmitter's output, the transmitted bitstream, will be received by the receiver. The reverse process is applied to the transmitted bitstream. The result is the reconstructed analog signal.
